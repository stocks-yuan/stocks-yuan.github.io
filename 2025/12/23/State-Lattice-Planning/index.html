动力学约束下的运动规划——State Lattice Planning
最近更新：2025-12-29   |   字数总计：1.1k   |   阅读估时：3分钟   |   阅读量：次

参考文献：知乎：State Lattice Planner(状态栅格规划)

一、整体思想

State Lattice Planner通过将状态空间离散化为晶格结构，为机器人规划符合运动学约束的路径。这种规划器需要解决两个关键问题：一是如何生成连接两个状态点的可行轨迹(满足动力学约束的)，二是如何在这些轨迹中找到最优路径。

第一个问题：需要考虑生成状态栅格的方式，若是控制空间采样，只需要根据当前状态和离散的控制量向前积分即可得到；若是状态空间采样，通过求解边界值问题（BVP）可得到最优轨迹。

第二个问题：通过控制空间采样或状态空间采样构建了满足机器人动力学约束的图后，求解图中两个状态点间的最优路径，其实就转变为了图搜索问题，可以使用A*等算法求解。

普通栅格(grid)与状态栅格的异同点： 相同之处：它们都是将连续空间离散化，简化路径规划。

不同之处：状态栅格中，每个顶点的连线都是根据机器人的运动学模型生成的，都是切实可行的路径，而普通栅格的方法并没有结合模型，所以两点之间的路径，机器人并不一定能通过，举个例子：

二、状态栅格生成

对于普通栅格点，直接给定栅格距离等，直接生成就好，而状态点阵需要结合机器人的运动空间去生成。有两种生成方法，分别为对控制空间的采样和对状态空间的采样。

2.1 控制空间采样与状态空间采样

控制采样的技术路线源自经典的运动学建模思想。这种方法将机器人的控制指令空间进行离散化，预设一组基础运动模式（如固定转向角、恒定速度等），通过前向积分生成候选路径。以差速驱动机器人为例，算法可能预设左转30度、直行、右转30度，三种基础控制指令，在规划时将这些指令按不同时长组合，形成扇形展开的候选路径集，如下图(a)所示。这种方法的优势在于物理可解释性强，易于求解。但其局限性同样显著：当环境障碍复杂时，由于缺乏目标导向，规划效率较低。

状态采样则直接在目标状态空间（如位置、朝向）中离散化采样点，通过求解边界值问题(BVP)，得到连接当前状态与目标状态的可行轨迹，如上图(b)所示。这种方法的优势在于解空间覆盖度高，能够生成形态多样的候选路径，特别适合结构化道路中需要精确贴合车道线的场景。但代价是计算复杂度显著增加，反向轨迹求解可能涉及大量迭代优化，实时性面临挑战。

2.2 Boundary Value Problem

OBVP（Optimal Boundary Value Problem，最优边界值问题）是最优控制中经典数值计算问题：两点边值问题。当采用基于状态空间的采样方式时，需要求解OBVP来生成任意两点之间符合运动学方程的最优轨迹。OBVP将轨迹规划问题转化为最优控制问题，通过Pontryagin极小值原理求解，得到满足边界条件的最优轨迹。
